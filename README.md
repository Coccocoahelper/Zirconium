# Zirconium
This is an Ornithe port of Eliotex's Zirconium mod, with additional optimizations.

Here are the things it currently optimizes:

- Chunk saving (from Eliotex's Zirconium mod)

- Language reloading (from Mixces' special mod)

Currently requires Java 21 or higher.
